# 高等學術論文寫作互動式教學 / Advanced Academic Writing Interactive Tutorial

**🌐 Live demo / 線上瀏覽**: <https://charlene717.github.io/academic-writing-interactive-tutorial/>


雙語 (zh / en) 互動式教程，聚焦生醫與生資領域的**碩博士學位論文**與**SCI/SSCI 期刊論文**寫作與發表流程。

## 結構 Structure

```
Academic_Writing/
├── academic-writing-interactive-tutorial-main/   ← 本資料夾：14 主題頁面 + index
├── references/                                    ← 參考資料佐證
└── academic-writing-quiz/                         ← 互動考題模組
```

## 14 個主題頁面 Topics

### 區塊一：研究與規劃 (Research & Planning)
1. **research-question.html** — 研究問題與假設 (PICO/FINER, hypothesis)
2. **literature-review.html** — 文獻回顧與引用策略 (narrative/systematic/scoping)
3. **structure-imrad.html** — IMRaD 與學位論文章節結構

### 區塊二：八股段落寫作 (Section-by-Section)
4. **introduction-writing.html** — 緒論寫作 (CARS model)
5. **methods-writing.html** — 方法寫作與可重現報告 (EQUATOR/STROBE/CONSORT/ARRIVE)
6. **results-figures.html** — 結果寫作與圖表設計
7. **discussion-limits.html** — 討論、限制與結論
8. **abstract-title.html** — 摘要、標題、關鍵字 SEO

### 區塊三：寫作技藝與引用 (Craft & Citation)
9. **style-language.html** — 學術寫作語言與風格 (active voice, hedging, L2 errors)
10. **citation-references.html** — 引用、參考文獻管理 (Zotero, EndNote, CSL)

### 區塊四：投稿與發表 (Submission & Publication)
11. **journal-strategy.html** — 期刊選擇與投稿策略 (IF, OA, predatory journals)
12. **peer-review-response.html** — 同儕審查與 response letter
13. **ethics-authorship.html** — 倫理、CRediT、ICMJE、ORCID
14. **ai-tools-reproducibility.html** — AI 寫作工具、preprint、可重現實踐 (2026 政策)

## 視覺設計 Design

- 配色：Oxford navy (#1e3a5f) + dark goldenrod (#b8860b)，學術典雅風
- 字體：Crimson Pro (襯線標題) + DM Sans (內文) + JetBrains Mono (程式)
- localStorage key：`acadwrite_lang` / `acadwrite_quiz_lang` / `acadwrite_quiz_progress_v1`

## 互動元素 Interactive Elements

- 🌳 決策樹 (Decision tree callouts)
- ✅/❌ 好壞範例對照 (Bad vs Good writing examples)
- ⚡ 互動模擬器 (期刊選擇器、IMRaD 段落判定、IF 計算等)
- 📊 Chart.js 視覺化
- 📝 每頁 3 題自我檢測

## 使用 Usage

直接打開 `index.html` 開始學習。右上角語言切換按鈕可在中/英之間切換。
